# Hybrid-Metaheuristic-Neural-Network-Optimization-for-Regression-on-Mining-Dataset
Benchmarking three hybrid metaheuristic-neural network models (HHO-MLP, SHO-ANN, GSO-MLP) for regression on a mining dataset. HHO-MLP achieved best performance with Normalized RMSE: 0.0316, MAE: 0.0264, R²≈1.0. Built with Python, NumPy, Scikit-learn.
